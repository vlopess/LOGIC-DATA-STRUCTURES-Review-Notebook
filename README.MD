# Logic & Data Structures

An introductory book on programming, logic, and data structures using Java.

## About

This book covers programming fundamentals from basic concepts to essential data structures, using Java as the teaching language. Content is presented progressively with practical examples and review exercises.

## Contents

- Basic programming concepts
- Compilation vs interpretation  
- Memory structures
- Syntax, semantics, and pragmatics
- Programming paradigms
- Control structures
- Arrays and ArrayLists
- String manipulation
- Basic algorithms
